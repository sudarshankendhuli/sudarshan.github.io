Title: : MODELING AND ANALYSIS OF ELECTRICAL POWER SYSTEM USING MATLAB/SIMULINK AND POWER WORLD

Description: Simulation software become more and more important in electrical system due to the advantages against the hands-on models, but the later remains also very useful due to the similitude with the real industry installation. The components are modelled using MATLAB/SIMULINK and POWER WORLD for the study and documentation. The advantage against the classical model is that here it can be modelled the lossless line. The voltage increase in this operating condition can become dangerous for the insulation of the line and in practice shunt inductive reactive compensation is done in order to avoid it. A simulation using many PI section line can be done if the user desire to measure the voltage and current in any section of the line. In this case, when measurement blocks on the diagram become numerous, MATLAB/SIMULINK give the results centralized in a list which is accessible using powergui button, and there is the possibility to export it in other useful files format like xls spreadsheets etc. the result of power flow and short circuit study are recorded and documented in tabular form for IEEE 3, 5 and 9 bus configuration. From these two software we find close similarity in their output result.
